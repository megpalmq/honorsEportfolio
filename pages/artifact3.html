Emerging Expertise: Web Design & Interactive User Experiences

Throughout my studies in Media Arts and Science, I have developed an emerging expertise in web design and front-end development. My work focuses on creating interactive, visually engaging digital experiences while maintaining strong usability and functionality. Through academic projects, an honors contract, and independent design work, I have strengthened my skills in user experience design, responsive development, and modern web technologies. The following projects demonstrate how I have applied these skills across different experiences while growing as a web designer and developer.

Project 1: Web of Designers Website

View Website 🔗

For this project, I designed and developed the Web of Designers website to highlight influential designers and their work. The goal was to create a visually engaging platform that presents information in a clear and interactive way. I focused on layout design, typography, and user navigation to ensure the site was both aesthetically appealing and easy to use. I built the website using modern web development tools and deployed it on Vercel to ensure fast performance and accessibility. This project strengthened my ability to translate design concepts into functional web interfaces while considering user experience and responsive design.

Beyond the visual design, I spent time structuring the content so each featured designer felt unique while still fitting within one consistent system. I experimented with spacing, card hierarchy, and interaction states to make browsing feel intuitive and polished. This required balancing creativity with usability, especially when presenting larger amounts of information without overwhelming the user.

I also focused on responsive behavior and performance optimization so the site remained clean and fast across desktop and mobile devices. Through this process, I improved my confidence with front-end architecture, component consistency, and visual storytelling through code.

Caption: Web of Designers site interface, built to present research on influential designers with clear hierarchy and navigation.

Project 2: Dream Team Web App

View Web App 🔗

As part of my Honors Contract experience, I designed and developed the Dream Team web application. This project focused on full-stack web development and allowed me to explore how design and functionality work together to create a complete digital product. I implemented Firebase for user authentication and data storage while building a responsive interface that works across devices. Through this experience, I gained hands-on experience integrating APIs, managing data, and developing a dynamic application from concept to deployment. The Dream Team project demonstrates my ability to combine front-end design with back-end functionality, which is an essential skill in modern web development.

My role included planning user flows, organizing data structure, and designing interfaces that could support real-time interactions. I worked through multiple iterations to improve clarity in navigation and make key actions easier for users to complete. Testing each feature from both the design and technical side helped me understand how small UI decisions can directly impact usability and retention.

This project also strengthened my debugging and problem-solving skills, since building a full-stack app required consistent troubleshooting across authentication, data handling, and responsive layout behavior. By the end of the project, I had a stronger understanding of how to move from idea to working product in a production-like environment.

Caption: Dream Team full-stack app interface demonstrating Firebase-based authentication and responsive interaction design.

Project 3: Cool Math Games Redesign

For an extra project in my Digital Storytelling course, I redesigned the Cool Math Games website to improve its visual design and user experience. My goal was to rethink how the site could better organize its content and create a more engaging interface for users. I focused on improving layout hierarchy, navigation clarity, and overall visual design. By redesigning an existing platform, I learned how to evaluate a website’s usability and identify opportunities for improvement. This project helped me develop stronger skills in user-centered design and reinforced the importance of balancing creativity with usability.

I began by analyzing the original experience and identifying friction points, including crowded navigation, inconsistent spacing, and weak visual hierarchy. Using that audit, I mapped a revised information structure and created wireframes that emphasized clearer pathways for discovery and play. I then translated those concepts into higher-fidelity Figma screens to test typography, color use, and component consistency.

What made this project especially valuable was learning how to redesign an existing brand without losing what users already recognize. It pushed me to think critically about modern UI standards, interaction patterns, and accessibility while still preserving the playful identity of the original platform.

Caption: Figma prototype for a Cool Math Games redesign, focused on stronger navigation flow and visual clarity.

Reflection

These three projects demonstrate my growing expertise in web design and interactive user experience development. Through the Web of Designers project, I strengthened my ability to design visually engaging websites that clearly present information. The Dream Team web app expanded my skills into full-stack development, allowing me to integrate back-end functionality with front-end design. Finally, the Cool Math Games redesign helped me think critically about how existing platforms can be improved through better layout, navigation, and user experience. Together, these experiences reflect my development as both a designer and developer. As I continue pursuing opportunities in web design and development, these projects represent the foundation of the technical and creative skills I will bring into my professional career.

What stands out most across these projects is how my process has become more intentional. Earlier in my college experience, I focused primarily on creating something that looked polished. Now, I approach each project with a clearer strategy that starts with user needs, content structure, and interaction goals before visual styling. This shift has made my work more effective and more aligned with real-world design expectations.
